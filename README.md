# DL-Project_Coronary-Heart-Disease-Predictor
A machine learning tool based on the Framingham Heart Disease Study    
The tool takes input as  
Gender  
Smoking Status  
Cigarettes per day  
Do you take Blood Pressure Medications  
History of stroke  
History of hypertension  
Are you Diabetic  
Total Cholestrol  
Systolic BP  
Diastolic BP  
Body Mass Index  
Resting heartRate  
Glucose
